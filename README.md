# VHDL

Guiding myself trough the hidden path of VHDL.

## Requirements

 * [ghdl](https://github.com/ghdl/ghdl)

### Instaling ghdl from source

```
git clone https://github.com/ghdl/ghdl.git
cd ghdl
./configure --prefix=/usr/local
make
make install
```

### Compiling Source

```
make
```
